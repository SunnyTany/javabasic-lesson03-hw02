# Робота з Git, GitHub.

ДЗ полягає в таких діях:

- Зареєструватися на Github. 
- На локальній машині, в папці проектів, створити проект My-App, де створити пакет app. 
- В пакеті app створити клас ***Main***, де реалізувати функціонал такого виведення в консоль: *"This is shop-online app"*.
- Забезпечити залиття цього проекту на свій Github. Далі модифікувати текст виводу на такий: *"Shop-online for customers"*.
- Залити виконаний проект на свій GitHub репозиторій.
